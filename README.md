🧪 DIGEMAPS - Sistema de Gestión y Validación de Muestras

Aplicación web desarrollada en equipo para la gestión integral de muestras de laboratorio, enfocada en la trazabilidad, validación de resultados y generación de reportes profesionales.

📋 Descripción

Este sistema permite gestionar el ciclo completo de una muestra de laboratorio, desde su registro hasta su validación final. Está diseñado para facilitar el trabajo de analistas y validadores, garantizando una visualización clara, organizada y confiable de la información.

⚙️ Funcionalidades
📥 Registro y visualización de muestras
👨‍🔬 Asignación de analistas
🔬 Consulta de análisis:
Físico-Químico
Microbiológico
👁️ Visualización detallada de resultados
✅ Validación o rechazo de muestras
🔁 Devolución de muestras para corrección
📄 Generación de reportes en PDF (3 páginas):
Información de la muestra
Análisis físico-químico
Análisis microbiológico
🔗 Arquitectura

La aplicación funciona como un frontend desacoplado, conectado a un backend mediante APIs REST, consumiendo datos en tiempo real para la gestión de muestras.

🛠️ Tecnologías utilizadas
⚛️ React
▲ Next.js
🎨 Tailwind CSS + shadcn/ui
🔐 Autenticación con JWT
🌐 Consumo de APIs REST
📄 jsPDF + html2canvas
